خوش امدید به focker

سرویس داکر

سرویس پروکسی رایگان داکر

با سرویس فوکر به راحتی می توانید ایمیج های داکر را بدون هیچ محدودیتی دریافت کنید.

استفاده در runtime :

بجای

                                  docker pull nginx

بزنید

                                  docker pull focker.ir/nginx

استفاده در کانفیگ داکر :

فایل زیر را ادیت کنید

/etc/docker/daemon.json

این مقدار را وارد کنید

                                  { "registry-mirrors": [ "https://focker.ir" ] }

سپس داکر را ریستارت کنید

                                  systemctl restart docker
